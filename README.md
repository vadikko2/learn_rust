# learn_rust

A personal learning project for practicing Rust.

This repository contains coding challenges from [Rustfinity tracks](https://www.rustfinity.com/tracks). Each challenge lives in its own Cargo crate with a task description, starter code, and tests.

## Running tests

From a challenge directory:

```bash
cd <challenge-name>
cargo test
```

Each challenge includes a `description.md` file with the task details.
